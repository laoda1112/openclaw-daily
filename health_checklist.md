# 轻量健康检查清单

> 覆盖：可用性 / 内容生成 / 发布链路

## 1) 可用性（Availability）
- [ ] 主页返回 200，响应时间 < 2s
- [ ] 关键页面（公告页、更新页）返回 200
- [ ] HTTPS 证书有效期 > 7 天
- [ ] 静态资源（CSS/JS）返回 200
- [ ] DNS 解析正常（A/AAAA 或 CNAME）

## 2) 内容生成（Content Generation）
- [ ] sitemap.xml 或 feed.xml 可访问
- [ ] 最新内容更新时间距当前 < 48 小时（可按业务调）
- [ ] 首页包含关键标识文本（如站点标题/品牌词）
- [ ] 关键栏目不为空（如“最新发布”至少 1 条）

## 3) 发布链路（Publish Pipeline）
- [ ] 最近一次构建/部署成功（CI 状态为 success）
- [ ] 最近一次部署时间距当前 < 24 小时
- [ ] 部署后站点版本号/提交 hash 可追踪（可选）
- [ ] 发布失败触发告警（邮件/IM/Webhook）

## 4) 告警通道（Alerting）
- [ ] 失败 >= 1 项触发告警
- [ ] 告警信息包含：失败项、站点 URL、时间、最近部署信息
- [ ] 有明确的降噪规则（如连续 2 次失败才告警）
