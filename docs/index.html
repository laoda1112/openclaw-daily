站点特点

  • 纯静态 HTML/CSS，运行时零常驻进程
  • 构建仅拷贝文件，几乎零依赖
  • 适配低资源环境与 CDN 托管

部署方式

  • Cloudflare Pages（推荐）
  • GitHub Pages / Netlify（备选）
  • 本地静态服务器（验收/预览）

内容扩展

  • 编辑 src/index.html 更新内容
  • 新增页面：直接添加 HTML 文件
  • 如需动态能力，接入 Serverless

部署说明

  1. 运行 ./build_site.sh 生成 dist/
  2. 运行 ./serve_local.sh 本地预览
  3. 运行 ./deploy_cf_pages.sh 部署到 Cloudflare Pages
